# Security Policy

## Supported Versions

Security updates will be backported for the latest minor release series.

## Reporting a Vulnerability

Please report security issues via GitHub Security Advisories or email security@[example].com.

## Security Best Practices

- Run the test suite and CI gates before deploying.
- Prefer the latest Rust stable toolchain.
- Avoid enabling optional features you do not need.

## Disclosure Policy

We follow coordinated disclosure. We will acknowledge receipt within 72 hours, provide an initial assessment within 14 days, and release fixes as soon as practical.
